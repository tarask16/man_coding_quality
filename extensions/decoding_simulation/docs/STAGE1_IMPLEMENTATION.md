# Этап 1. Каркас изолированного расширения

## Реализовано

- самостоятельный пакет `manual_coding_sim_decoding`;
- локальный `pyproject.toml`;
- проверяемая YAML-конфигурация;
- безопасные пути вывода внутри папки расширения;
- контракт адаптера к публичным интерфейсам `manual_coding_sim`;
- проверка неизменности baseline-файлов по SHA-256;
- CLI для отображения конфигурации и проверки совместимости;
- тестовый модуль этапа 1.

## Не реализуется на этом этапе

- формирование кодированного сообщения `C`;
- ошибки фактического кодирования;
- обратная процедура декодирования;
- восстановленное сообщение `M'`;
- сквозные показатели качества;
- новые CSV-артефакты.

## Критерий завершения

Этап считается закрытым после локального подтверждения, что:

1. пакет импортируется из `extensions/decoding_simulation/src`;
2. YAML-конфигурация проходит проверку;
3. контракт с базовым пакетом имеет статус `is_compatible = true`;
4. список измененных baseline-файлов пуст;
5. тесты этапа и полная регрессия пройдены.
