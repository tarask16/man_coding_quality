# Корректировка этапа 16 — рисунок 4.2

Исправлена формулировка методического примечания в генераторе
`figure_4_2_token_frequency_and_filtering.py`.

## Причина

Локальный тест проверяет наличие точной строчной подстроки `отброшенные` в SVG.
В предыдущей версии использовалось слово `Отброшенные` с прописной буквы, поэтому
проверка была чувствительна к регистру и завершалась ошибкой.

## Исправление

Формулировка заменена на:

> Сведения об отброшенных до сохранения кандидатах в dictionary.json отсутствуют...

PNG и SVG пересобраны. Тесты в рабочем окружении не запускались; они выполняются
локально пользователем.
