# Этап 29. Рисунок 6.3 — распределение абсолютной ошибки

## Назначение

Генератор строит диссертационный рисунок по значениям интегрального априорного индекса и фактического интегрального качества. Для каждого сценария вычисляется:

\[
|e_i| = |Q_{pred,i} - Q_{fact,i}|.
\]

Рисунок включает:

- гистограмму абсолютной ошибки;
- эмпирическую функцию распределения;
- покрытие фиксированных порогов 0,05; 0,10; 0,15; 0,20; 0,30;
- MAE, медиану, RMSE, квартильные показатели и максимальную ошибку;
- идентификаторы сценария с максимальным абсолютным отклонением.

## Входные данные

- `reports/chapter5/q_pred.csv`;
- `data/processed/quality_targets.csv`.

Объединение выполняется по `scenario_id` и `protocol_id` с обязательной связью `one_to_one`.

## Методическое ограничение

Абсолютная ошибка не сохраняет направление смещения. Её распределение характеризует величину расхождения на вычислительном корпусе, но не доказывает абсолютную калибровку, вероятностное покрытие или переносимость модели.
