# Корректирующий пакет после локальной проверки этапа 27

Исправлены две регрессионные ошибки, выявленные полным локальным запуском тестов генераторов рисунков.

## 1. Рисунок 5.2

В SVG теперь гарантированно присутствует точная подпись `процедурная трудоёмкость`, ожидаемая тестом `test_png_and_svg_are_dissertation_ready`.

Дополнительно устаревший аргумент Matplotlib `vert=False` заменён на `orientation="horizontal"`, что устраняет `MatplotlibDeprecationWarning` для `boxplot`.

## 2. Рисунок 5.5

Заголовок правого маргинального распределения изменён так, чтобы SVG содержал точную подстроку `Распределение`, ожидаемую тестом.

## 3. Предупреждение теста рисунка 5.4

Регулярное выражение в `test_figure_5_4_q_pred_distribution.py` преобразовано в raw-строку. Это устраняет `DeprecationWarning: invalid escape sequence`.

## Проверка в рабочем окружении

PNG и SVG рисунков 5.2 и 5.5 повторно сформированы в 300 dpi. Проверено наличие всех строк, перечисленных в двух ранее упавших тестах. Pytest в рабочем окружении не запускался согласно принятому правилу проекта.
